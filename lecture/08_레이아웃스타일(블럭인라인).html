블럭, 인라인

블럭 요소 : 한 줄 전체를 차지하는 요소. 한 줄에 여러 요소가 올 수 없는 요소.
Ex. div, hn, p, hr, ul, ol, form, table, .. 등등
인라인 요소 : 한 줄에서 일부분만 차지하는 요소. 한 줄에 여러 요소가 올수 있는 요소.
Ex. span, img, input, label, buttom, textarea, br, .. 등등

화면 배치 방법 변경

블럭 요소와 인라인 요소의 속성을 변경하여 배치 방식을 변경할 수 있다.

블럭 요소를 인라인 요소로 변경

display : inline 테스트
인라인 요소로 변경되면서 width와 height 속성이 무시 된다.

인라인 요소를 블럭 요소로 변경

display : block 테스트
인라인 요소를 블럭 요소로 지정하면 width와 height 값을 지정할 수 있다.
