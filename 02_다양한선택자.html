속성 선택자

기본 선택자 뒤에 []를 사용하여 속성과 속성 값을 사용한다. 선택자[속성=속성값] { 설정내용 } 의 형식으로 사용한다.

자손/후손 선택자

자손 선택자 : 바로 아래의 요소, 후손 선택자 : 하위 요소 전부. 자손 선택자는 선택자>자손선택자 { 설정 내용 } 의 형식으로 사용하고 후손 선택자는 선택자 후손선택자 { 설정 내용 } 의 형식으로 사용한다.

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용한다. 바로 뒤에 있는 요소 선택 시 : 선택자 + 선택자 { 설정 내용 } 의 형식으로 사용하고 뒤에 있는 모든 요소 선택 시 : 선택자 ~ 선택자 { 설정 내용 } 의 형식으로 사용한다

반응 선택자

사용자의 움직임에 따라 달라지는 선택지. 사용자가 클릭한 태그 : 선택자:active { 설정 내용 }의 형식으로 사용하고, 사용자의 마우스가 올라간 태그 : 선택자:hover { 설정 내용 }의 형식으로 사용한다.

상태 선택자

입력한 양식의 상태에 따라 선택되는 선택자. 체크 상태의 input 태그 선택 : input:check { 설정 내용 }의 형식으로 사용한다.

초점이 맞추어진 input 태그 선택 : input태그:focus { 설정 내용 }의 형식으로 사용한다.

사용 가능한 input 태그 선택 : input태그:enabled { 설정 내용 }의 형식으로 사용가능하며, 사용 불가능한 input태그 선택 : input태그:disableed { 설정 내용 } 의 형식으로 사용한다.
